# MVP Specification

## Goal

Build an AI assistant capable of helping blind users understand their surroundings.

---

## Core Features

- Object Detection

- OCR

- Speech Recognition

- Voice Output

---

## Success Criteria

The user should be able to:

✔ Point the camera

✔ Ask:

"What am I looking at?"

✔ Receive a spoken answer.

---

## Out of Scope

- Face recognition

- Indoor navigation

- Autonomous navigation

These will be developed later.
