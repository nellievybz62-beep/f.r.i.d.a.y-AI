# Project FRIDAY Mobile App

## Overview

The mobile application is the first implementation of Project FRIDAY.

The smartphone performs AI processing while communicating with smart glasses for camera input and audio output.

---

## MVP Features

- Camera Preview
- Object Detection
- OCR
- Speech Recognition
- Text-to-Speech
- Voice Assistant
